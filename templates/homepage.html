Todo List API

This is a simple API to perform all the functions of a basic todo app.

The API is built on the Django framework and uses the Django Rest Framework.

Features

  • Register User
  • Login User
  • Create a todo
  • Update a todo
  • Delete a todo
  • Get all todos
  • Filter todos
  • Search todos
  • Order todos
  • View Documentation on ReDoc
  • View Documentation on SwaggerUI
